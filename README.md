 AI Smart Parking Guidance System

This project uses Computer Vision and YOLOv8 to detect available and occupied parking spaces in real time

Project Features
- Real-time parking occupancy detection
- Smart parking guidance system
- Occupied and vacant parking classification
- AI-based object detection using YOLOv8

Dataset
Parking Space Detection Dataset from Kaggle

Model
YOLOv8 Nano (Fine-Tuned)

Results
- Precision: 0.84
- Recall: 0.83
- mAP50: 0.90

Technologies Used
- Python
- YOLOv8
- OpenCV
- Kaggle
- Matplotlib
